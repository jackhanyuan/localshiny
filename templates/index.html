LocalShiny

Auto deploy Shiny apps locally

The LocalShiny is an open-source kit that can help developers share their Shiny applications online and help users install these Shiny applications locally. It includes the LocalShiny Web, the LocalShiny R package (for developers), and the LocalShiny Software (for users).

Features include:

  • Security: Install Shiny applications locally, so you don’t have to worry about privacy leaks.
  • Automatically Install: Even if you don't know anything about the program, you can easily install the Shiny app with LocalShiny Software without having to install any dependencies manually.
  • Easily Share: LocalShiny web serves as a platform for developers to share shiny applications built in R. With the LocalShiny package, developers create clones of applications and share the applications in the cloud easily.
